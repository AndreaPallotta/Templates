# Rust (Axum) HTTP server template

## Requirements

| Requirement  | Version |
| ------------ | ------- |
| Cargo (Rust) | 1.69    |

---

## Setup

1. Clone the repository.

    ```bash
    git clone git@github.com:AndreaPallotta/Templates.git
    ```

2. Navigate to the rust directory

    ```bash
    cd Template/rust
    ```

3. Run API

    ```bash
    cargo run
    ```

4. Build API

    ```bash
    cargo build
    ```
